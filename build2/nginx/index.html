Mediaserver quick access

Plex
The mounted folders in plex will be available under /mnt/
Couchpotato
The mounted folders in plex will be available under /mnt/ and the Deluge service will be availble at host: deluge:58846 using username: deluge and password: deluge. Use directory /mnt/your_folder/Film etc. to download to the right location.
Sonarr
The mounted folders in plex will be available under /mnt/, deluge is available at host: deluge and use the password from the Deluge web ui.
Deluge
Default password is deluge. It is important to change the following two settings: Enable the Label plugin under Preferences/Plugins and enable Allow Remote Connections under Preferences/Daemon
Jackett
Jackett can be used in Sonarr and Couchpotato, add it as Torznab or TorrentPotato
